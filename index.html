About AAS

What is AAS?

AAS is a web-based academic management tool developed by Agunextech. It helps schools manage classes, enter student marks, generate instant results and produce professional documents, all from one place.

🎓
CBC and 8-4-4 Ready
Supports both curricula, including CBC career path selection for Grade 10 and above.
🤖
AI Powered
Generate personalised AI remarks for individual students or the full class in one click.
🛡️
Verified Documents
Every certificate and report card carries a unique verification code. Forgery is impossible.
📲
WhatsApp and SMS
Send results directly to parents, one by one or in bulk, via WhatsApp or SMS.
AAS Features

Everything in One System

  • Mark entry with auto-save every 1.5 seconds. Nothing is ever lost.
  • Results sorted highest to lowest the moment marks are saved.
  • AI remarks generated per student or for the full class.
  • Report cards and merit lists, ready to download and print.
  • Certificates for the top 3 students, complete with a verification code.
  • WhatsApp and SMS delivery to parents, individually or in bulk.
  • Custom mark sheet generator for CBC and 8-4-4 classes.
  • Data backup and restore using a simple JSON file.
  • Global search across all students, classes and records.

Students

Adding Students

There are three ways to add students. Swipe through to see each method.

✏️
Add One by One
Manually enter each student
All Plans
🖼️ Screenshot: Add Student Form screenshots/09a_add_student.jpg

Tap the + Add Student button to open the student form. Fill in the name, admission number, gender and parent phone number, then save. Repeat for each student. Best suited for small classes or when adding a few new learners mid-term.

📂
Upload CSV File
Import a full class at once
Starter and Pro
🖼️ Screenshot: CSV Upload screenshots/09b_csv_upload.jpg

Prepare a spreadsheet with columns for name, admission number, gender and parent contact. Save it as a CSV file and upload it directly into AAS. The whole class loads in seconds. AAS flags any rows with errors so you can fix them before confirming the import.

📷
AguScan AI
Photograph a handwritten mark sheet
Pro Only
🖼️ Screenshot: AguScan screenshots/09c_aguscan.jpg

Take a clear photo of your handwritten mark sheet. AguScan reads the names, admission numbers and scores, then shows you a preview before anything is saved. Correct any misread name or mark right there, then confirm. No typing required.

  • 1Take a photo of the handwritten sheet.
  • 2AguScan reads and displays the data for review.
  • 3Fix any errors, then confirm to import.
